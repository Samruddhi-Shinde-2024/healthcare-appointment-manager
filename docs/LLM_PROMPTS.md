# LLM Prompts

## Pre-Visit Summary

Purpose

Generate a concise doctor-facing summary before the consultation.

Input

- Patient symptoms
- Appointment details

Output

- Chief complaint
- Possible concerns
- Suggested follow-up questions
- Urgency indication

---

## Post-Visit Summary

Purpose

Generate patient-friendly follow-up instructions.

Input

- Clinical notes
- Doctor notes
- Prescribed medicines

Output

- Medication schedule
- Follow-up guidance
- Simplified patient instructions
